Быстрое создание интернет магазина, интегрированного с платформой АвтоОфис

== Description ==

Создание интернет магазина на базе платформы WordPress интегрированного с сервисом АвтоОфис: http://autoweboffice.com/

== Installation ==

= Установка =

1. Загрузите плагин "AutoWebOffice Internet Shop" (АвтоОфис - Интернет магазин) на ваш блог
2. Активируйте загруженный плагин
3. В правом вертикальном меню выберите пукт "Магазин"
4. Из выпадающего списка меню плагина выберите раздел "Настройки"
5. Укажите настройки подключения к вашему магазину, зарегистрированному в сервисе АвтоОфис: http://autoweboffice.com/
6. Сохраните введенные данные
7. После ввода данных для подключения к магазину, в разделе "Настройки" нажмите на кнопку "Обновить товары"
8. Дождитесь завершения обновления
9. В левом боковом меню выберите раздел "Магазин->Справка"
10. Следуйте дальнейшим инструкциям

Если появятся вопросы или предложения по улучшению плагина, отправляйте их на почту zakaz@autoweboffice.com

= Рекомендации =

* PHP 5.1 или выше
* WordPress 3.9.1 или выше

== Frequently Asked Questions ==

= Где я могу найти документацию по работе с плагином интернет-магазина? =

Сразу после установки плагина к вам на сайт, в левом боковом меню выберите раздел "Магазин->Справка". 

= Будет ли работать плагин интернет-магазина с моей темой? =

Да, плагин интернет магазина будет работать с любой темой, но может потребоваться дополнительная стилизация.
Если возникнут проблемы, обязательно свяжитесь с нами по почте: zakaz@autoweboffice.com

= Если мне не будет хватать возможностей плагина интернет-магазина, могу ли я обратится к вам за доработками? =

Да, конечно! Мы накапливаем пул обращений и пожеланий, и регулярно реализуем топовые пожелания в новых версиях плагина.
Свои пожелания отправляйте к нам на почту: zakaz@autoweboffice.com

= Что такое АвтоОфис? =

АвтоОфис - это современная платформа для ведения бизнеса в Интернете. АвтоОфис позволяет автоматизировать все функции обычного бизнеса: выставление счетов, прием платежей, маркетинг, управление сотрудниками и многое-многое другое.
Именно с данной платформой взимодействует плагин интернет-магазина.

Более подробно почитать о возможностях сервиса, вы можете пройдя по ссылке: https://autoweboffice.com/
